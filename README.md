# Mega Man X Inspired Prototype

Gameplay/combat prototype inspired by Mega Man X, developed in Godot using a modular and scalable architecture.

The project was built as a large-scale gameplay and systems study, focusing on reusable systems, combat orchestration, visual layering, and structured game architecture.

---

# Features

## Gameplay Systems
- X movement system
- Dash mechanics
- Shooting system
- Combo attack system
- Boss battle system
- Armor system
- Stage select
- Character select
- Health and combat management

## Technical Systems
- Custom particle system
- Layered sprite rendering system
- Modular gameplay architecture
- Decoupled managers and systems
- GameManager orchestration
- Global SoundManager
- Data-driven organization
- Reusable FX systems
- State-based gameplay logic

---

# Technologies

- Godot Engine
